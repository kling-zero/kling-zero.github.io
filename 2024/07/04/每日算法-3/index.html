最长回文子串

给你一个字符串 s，找到 s 中最长的 回文 子串。

思路分析

  • 使用动态规划
  • 两个for循环，一个j负责遍历字符串，一个i从头开始遍历到i,dp[i] [j]表示 从i到j是否为回文子串
  • dp[i] [j]=dp[i+1] [j-1]
  • 如果间隔小于3 也是也是回文子串

  • cur=next

搜索旋转排序数组

思路分析

  • 使用二分查找，确定前半部分，还是后半部分

  • nums[mid] 与num【l】比较，判断在哪个部分

  • if (nums[mid] >= nums[l]) {
    if (nums[l] <= target && target < nums[mid]) {
        r = mid - 1;
    } else {
        l = mid + 1;
    }
} else {
    if (nums[mid] < target && target <= nums[r]) {
        l = mid + 1;
    } else {
        r = mid - 1;
    }
}

岛屿数量

思路分析

  • dfs（）函数，遍历上下左右的位置
  • inArea判断是否还在方格中
  • 判断是否存在陆地，并设置为2，如果递归一次，表示所有相邻的都遍历了并设置为2了
  • 主函数循环遍历，遇到1则dfs，然后count++
